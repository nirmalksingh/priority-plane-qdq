This is ATC Subsystem to demonstrate that the planes' queue request are maintained in a data structure and they are dequeued such that they are dequeued in the following order:
1st EL=Emergency Large, 2nd ES= Emergency Small
3rd VL = Vip Large, 4th VS= Vip Small.
5th CL = Cargo large, 6th CS = Cargo small.
Such that: a passenger small plane will have priority to land even if it is queued after Cargo Large etc.

Add test planes (Press the following button) to see their add sequence and dequeue sequence which is as follows:
EL, ES, VL, VS, PL,PS, CL,CS

Enter planes to queue manually(Press the following button), the dequeue sequence is shown at in the next life at button press.
