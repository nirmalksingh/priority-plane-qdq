This is ATC Subsystem to demonstrate that the planes' queue request are maintained in a data structure and they are dequeued in the following order:
1st EML=Emergency Large, 2nd EMS= Emergency Small
3rd VPL = Vip Large, 4th VPS= Vip Small.
5th PGL = Passenger Large, 6th PGS= Passenger Small.
7th CGL = Cargo large, 8th CGS = Cargo small.
Such that: a passenger small plane will have priority to land even if it is queued after Cargo Large etc.

Add test planes (Press the following button) to see their add sequence and dequeue sequence

OR

Enter planes to queue manually(Press the following button), the dequeue sequence is shown in the next line on button press.

Only valid inputs are: EML, EMS, VPL, VPS, PGL, PGS, CGL, CGS for emergency large/small etc.

-the following shows the dequeue order(top to bottom) confirming to the given priority order above.
-the first two letters indicates the type and size of plane(EL is Emergency Large).
-and the number that follows indicates when that plane was put in the queue.
-each type of plane (among the 4 types) is assigned their own sequence number for when it was put in the queue.
